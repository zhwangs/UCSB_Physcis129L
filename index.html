Introduction to Scientific Computing

Physics 129AL

Fall 2023

Broida Hall, 5223

Course Description

This course is a physics orientated survey of basic concepts in modern computation, and the following outline is subject to change. 1) Foundations of modern computation, hardware and software, input/output (I/O), UNIX operating system (os), UNIX shell and file management, bash script, version control, network, data acquisition (DAQ)*. 2) An introduction to Python, basic programming, data structures, read/write, functions, scoping*, objects, error handling*, debugging*, pipe*, parallel computing. 3) Important Python packages, os, Numpy, Matplotlib, Scipy*. 4) Common statistical methods in physics, random process, probability density function (PDF), central limit theorem, Bernoulli process, binomial process, Poisson process, Lorentz (Cauchy) distribution. 5) Common computation techniques in physics, discrete Fourier transform, numerical integration and differentiation, implicit and explicit iterative methods for differential equations, gradient descent, stochastic sampling, Monte Carlo, Markov chain, PID control, machine learning, and deep learning. 6) Applications in physics, Electrostatics, Diffusion, Brownian motion, driven system, hydrodynamics, phase transitions, molecular dynamics, ab initio approaches to electronic structure, quantum state (Qubit) evolution, numerical renormalization group.

This course is not a theoretical physics course, but rather a survey of modern computation in physics. Therefore, I will provide all necessary physics background/concepts to you along the way. However, students are expected to have basic knowledge in classical mechanics, quantum mechanics, and statistical mechanics. By the end of this course, you will be asked to complete an independent project, choosing from selected physics topics.

Outline

Week 1 Introduction to the course and software installation. Basics on Linux shell and file management. (Problem set 1)

Week 2 Bash script and version control. Network protocols, Introduction to Python . (Problem set 2)

Week 3 Introduction to Python, continue. Introduction to Python packages. (Problem set 1 due) (Problem set 3)

Week 4 Introduction to statistical methods in physics. Statistical methods in physics, continue. (Problem set 2 due) (Problem set 4)

Week 5 Numerical integration and differentiation, implicit and explicit iterative methods for solving differential equations. Gradient descent, stochastic sampling, Monte Carlo, Markov chain, PID control. (Problem set 3 due) (Problem set 5)

Week 6 Discrete Fourier transform, machine learning, and deep learning. Electrostatics: Laplace's equation and Poisson's equation. (Problem set 4 due) (Problem set 6)

Week 7 Simple diffusion. Driven systems. (Problem set 5 due) (Final Project)

Week 8 Brownian motion. Phase transition. (Problem set 6 due)

Week 9 Density functional theory. Quantum state (Qubit) evolution.

Week 10 umerical renormalization group. Final Review. (Final project due)
