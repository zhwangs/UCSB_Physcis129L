Get Started

Oracle VM VirtualBox and Ubuntu

VirtualBox runs on Windows, Linux, macOS, and it is a powerful tool to access and explore various operating systems. To install the Oracle VM VirtualBox, visit //www.virtualbox.org/wiki/Downloads. Make sure to check the version. After the downolads, you need to specify the ISO image, for Ubuntu Desktop, visit https://ubuntu.com/download/desktop. We will use Ubuntu 22.04.3 LTS.

.

Step 1: Shown in the right image. Open the VirtualBox Manager, and click New. Enter a Desktop name, and ISO image. Make sure to select Linux and Ubuntu 22.04.3 LTS, Jammy Jellyfish. Check Unattended Installation.

.

Step 2: Allocate CPU cores, RAM, and disk usage. Make sure the ram is larger than 2GB. In addition, you need at least 15.7GB disk space to install Ubuntu.

.

Step 3: Run it in the VirtualBox Manager, and enter "try or install Ubuntu". Following the instructions.

Anaconda Distribution

In the remaining of the course, we will use Jupyter notebooks for completing problem sets. Conda environment has great supports on Python. I recommend Anaconda Distribution simplely because it runs on Windows, Linux, macOS. Visit https://www.anaconda.com/download#downloads to download. If you are not familar with Ubuntu, you are welcome to download it via the graphical installer on Windows or MacOS.
